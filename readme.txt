Integrate AI-powered customer support into your WooCommerce store with Laxi AI chatbot.

== Description ==

Laxi AI for WooCommerce brings intelligent customer support to your online store. The plugin seamlessly integrates with your WooCommerce installation to provide automated customer support through an AI-powered chatbot.

= Features =
* Easy setup and configuration
* AI-powered responses
* Custom knowledge training
* Personalized chatbot appearance
* Real-time customer support
* WooCommerce integration

= Requirements =
* WordPress 5.0 or higher
* WooCommerce 3.0.0 or higher
* PHP 7.2 or higher

== Installation ==

1. Install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Laxi AI menu item in your admin menu to configure the plugin
4. Follow the setup steps to connect your store and customize your agent

== Frequently Asked Questions ==

= Do I need a Laxi AI account? =

No, the plugin will automatically create an account for you

= Is this plugin free? =

Yes! The plugin itself is free, and Laxi.ai offers a generous free starter plan. For additional features like expanded agent knowledge and increased message limits, premium subscription plans are available.
